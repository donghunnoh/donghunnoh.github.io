Manipulation with a Legged Mobile Robot

New project that I am currently working on
Project Overview
We are developing a mid-size humanoid robot using proprioceptive actuators. As this project is in the early stages of development, we are experimenting with various approaches.
My Contributions
Based on my experience with dual-arm manipulator control from Project YORI, I am currently working on developing a whole-body controller for this humanoid robot, preparing it for interactions with people. Additionally, I am broadening my field to loco-manipulation, focusing on developing a state estimator and a basic footstep planner to enable the robot to walk.
